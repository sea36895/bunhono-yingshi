# DESIGN.md

## 气质与意象
深夜家庭影院：关闭主灯后仅剩屏幕微光的客厅，深色墙面吸收多余反光，视频封面是唯一色彩焦点。导航与文字如遥控器按键般克制、清晰、随时可触达。

## 视觉策略
- 图像方向：视频封面以海报比例 2:3 为主，保持原始色彩不做滤镜处理
- 图形语言：圆角矩形卡片 + 无衬线文字，信息密度优先于装饰

## 配色方案
- 主色：#09C878（遥控器绿灯 — "播放"隐喻，源自参考文件的核心品牌绿）
- 强调色：#398439（深绿 — 悬停/选中态，表示"已激活"）
- 背景：#F6F5F6（浅灰暖调 — 墙面微光，降低视觉疲劳）
- 卡片：#FFFFFF（纯白 — 内容容器，与背景分离）
- 文字主色：#333333 / 辅色：#666666 / 标签：#8c8c8c

## 字体排版
- 字体族："Segoe UI", "Microsoft YaHei", "Hiragino Sans GB", sans-serif（系统字体栈，无需外部加载）
- 标题：24px / 正文：14px / 辅助：12px
- 行高节奏：1.4–1.6

## 页面结构
- **Header**：Logo + API源切换下拉 + 搜索框 → 导航栏（分类横滚）
- **内容区**：视频网格（5列PC / 3列移动）或详情页（海报+信息+播放列表）
- **Footer**：站点名 + 联系方式
- **分页**：居中简洁，首页/上页/下页/尾页

## 交互与状态
- 卡片悬停：微阴影抬起（box-shadow 过渡 0.3s）
- 链接悬停：变为主色 #f60 橙色（参考文件既有交互）
- 分类选中：背景 #398439 深绿
- API源切换：Cookie 持久 + 即时刷新

## 设计禁忌
- 不使用渐变色按钮（影视站内容为王，按钮不应抢焦点）
- 不引入外部图标库（用文字标签即可，减少加载）
- 不在首屏使用懒加载（首屏图片必须立即可见）
